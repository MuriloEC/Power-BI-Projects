# 📊 Dashboard de Análise de Vendas Multilojas

Este repositório contém um projeto de Business Intelligence desenvolvido no Power BI para análise de desempenho de vendas de diferentes unidades e produtos ao longo do tempo.

## 🎯 Objetivo do Projeto
O objetivo deste painel é monitorar a evolução das vendas, permitindo identificar tendências, sazonalidades e o desempenho individual de produtos em 5 lojas distintas (Loja 01 a Loja 05).

## 🛠️ Tecnologias e Ferramentas
* **Power BI:** Construção dos visuais e interatividade.
* **Power Query:** Tratamento da base de dados, incluindo o processo de *unpivot* para transformar colunas de datas em atributos de linha, facilitando a análise temporal.
* **Excel/CSV:** Fonte de dados original.

## 💡 Insights e Funcionalidades
1.  **Visão Temporal:** Gráficos que demonstram a variação das vendas.
2.  **Filtros Dinâmicos:** Possibilidade de segmentar os dados por Loja ou por Produto específico para análises granulares.
3.  **Matriz de Desempenho:** Tabela detalhada com a performance cruzada entre unidades e produtos.
